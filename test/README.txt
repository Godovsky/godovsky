A program for copying information about the memory used from the /proc/meminfo file, 
removing all colon characters from it and sorting data in descending order.
The result is saved to the root folder of the program.
I didn't bother with swapping the nodes of the list and just swapping their data.
To compile the program, you need to run the terminal in the root folder with the source code
and execute the command "bash mkfile.sh ". After that, the file "execute" will appear in the folder,
which can be run with the command "./execute".

Программа для копирования информации об используемой памяти из файла /proc/meminfo,
удаления из него всех символов двоеточия и сортировки данных по убыванию.
Результат сохраняется в корневую папку программы.
Не стал заморачиваться со сменой местами узлов списка и просто меняю местами их данные.
Для компилирования программы необходимо в корневой папке с исходным кодом запустить терминал
и выполнить команду "bash mkfile.sh". После этого в папке появится файл "execute",
который можно запустить командой "./execute".
